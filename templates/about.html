This application provides a detailed and easy-to-use map based on data from OpenStreetMap project, a publicly available geographic database with global coverage. Check out www.openstreetmap.org for more information and to contribute to the project.

The map covers the box ranging from ({{config.extent[0]}}, {{config.extent[1]}}) to ({{config.extent[2]}}, {{config.extent[3]}}) - WGS84 coordinate projection system.
The data displayed has been updated on {{ config.data_update }}.

The information contained in the application might not have the same level of accuracy for all locations shown on the map. If you find inaccuracies and problems with the map or points of interest and want to contribute, you can visit www.openstreetmap.org and edit the map to update it.

The application will be regularly updated to reflect the data from the OpenStreetMap database, but as we can't guarantee the perfect accuracy of the information, the application is not intended be used as a navigator.
